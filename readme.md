**Description:**\
\
A Command Line Interface (CLI) Python game where the user attempts to
guess a randomly generated number within a customisable range.\
\
The user selects a minimum and maximum value, and the program generates
a random number within the specified range. The game continues until the
user correctly guesses the number, while simultaneously tracking the
number of attempts.\
\
**Concepts Demonstrated:**\

-   \- Functions and modular code structure

-   \- While loops

-   \- Conditional logic

-   \- Input validation

-   \- Exception handling

-   \- Random number generation

-   \- State tracking\
    \
    **How to Run:**\

1.  Ensure Python is installed (Python 3 is recommended).

2.  Clone the repository or download the file.

3.  Run the script:\
    \
    \`\`\`bash\
    python3 Number_Guessing_Game.py\
